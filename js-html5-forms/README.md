# Comprehensive HTML5 Form Validation

A single-page web application demonstrating a wide variety of HTML5 input types, built-in form validation, and DOM manipulation using Vanilla JavaScript.

## Features

* **Rich Input Types:** Utilizes various HTML5 inputs including text, password, radio buttons, checkboxes, date picker, color picker, dropdowns (`select`), and auto-complete lists (`datalist`).
* **Input Synchronization:** The range slider and number input for "Height" are bound together with JavaScript, updating each other in real-time.
* **Client-Side Validation:** * Enforces character limits, patterns, and required fields directly via HTML attributes.
  * Custom JavaScript validation to ensure passwords match before submission.
* **Dynamic Data Display:** Intercepts the default form submission (`event.preventDefault()`) and dynamically renders the collected form data securely on the page.
* **Zero Dependencies:** Runs entirely on standard HTML, inline CSS, and plain JavaScript.

## How to Run

Since this is a static, standalone file, no server setup is required. Simply download the `.html` file and double-click it to open it in any modern web browser.
